# tools/

This directory will contain build tools, code generators, analysis utilities, and development infrastructure for the Reikan project.

## Planned Contents

- `tools/build/` — Build system scripts and configuration
- `tools/codegen/` — Code and stub generators (e.g., from device description DSL)
- `tools/lint/` — Linters and static analysis tools
- `tools/trace/` — Trace and audit log analysis tools
- `tools/sim/` — Simulation and emulation harness utilities
- `tools/bench/` — Benchmarking infrastructure

## Status

No tools exist yet. Build infrastructure will be established during Phase 1 (language) and Phase 2 (boot chain).

See [docs/architecture-outline.md](../docs/architecture-outline.md) §7 and §23 for relevant design outlines.
