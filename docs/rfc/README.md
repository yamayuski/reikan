# docs/rfc/

This directory contains Requests for Comments (RFCs) for the Reikan project.

## What is an RFC?

An RFC is an open proposal on a specific design question, soliciting feedback from contributors before a decision is made. RFCs may be revised as discussion progresses. Once an RFC reaches consensus, it is either accepted (producing an ADR) or rejected.

## RFC Format

```
# RFC-NNNN: Title

**Status:** Open | Accepted | Rejected | Withdrawn

## Summary

One-sentence description of what this RFC proposes.

## Motivation

Why is this change needed?

## Proposal

Detailed description of the proposed design.

## Alternatives

What alternatives were considered?

## Open Questions

What needs to be resolved before this can be accepted?
```
