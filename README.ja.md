# 黎環 / Reikan

> **AI-First OS — 次世代オペレーティングシステム**

---

## 概要

**黎環（れいかん / Reikan）** は、AI-Firstオペレーティングシステムの、ゼロからの設計・実装を目指す初期フェーズのアーキテクチャプロジェクトです。

「黎（れい）」は夜明け前の暗闇・夜明けを意味し、「環（かん）」は輪・循環・交換を意味します。AIエージェント・異種計算資源・機械間協調が第一級市民となる、コンピューティングの新しい循環の幕開けを表しています。

このプロジェクトは Linux・Windows・macOS のフォークではなく、それらのアプリケーションモデル・システムコールインタフェース・設計上の前提を一切引き継ぎません。次の問いに答えるために作られています：

> *AIエージェントが人間のアプリケーションではなく計算の主役であるとき、オペレーティングシステムはどのような姿をとるべきか？*

---

## プロジェクトビジョン

Reikan は以下を根本目標とする **ケイパビリティ中心・エージェントネイティブ・形式検証可能なOS** です：

- AIエージェントがOSの第一級市民であり、メモリ・目的・ツール・ライフタイムをカーネルが管理する
- 異種計算資源（CPU・GPU・NPU・TPU・DMAエンジン）を統一されたリソースファブリックとしてスケジューリング・オーケストレーションする
- 安全性・監査可能性・検証可能性をレガシー互換性より優先する
- 人間とLLMの両方が記述しやすいよう設計された新しい高水準システム言語によってCを置き換える
- 人間と機械の知性が、共有された・検査可能な・ポリシーで管理された実行環境を通じて協調する

## 設計原則

| 原則 | 説明 |
|---|---|
| **AI-First** | エージェント・モデル・推論パイプラインはOSのネイティブ構造体であり、ユーザー空間の後付けではない |
| **ケイパビリティ中心** | すべての権限は明示的・委譲可能・監査可能。環境的（ambient）なアクセスは存在しない |
| **検証可能性** | 信頼されたコンピューティングベース（TCB）は小さく、形式仕様化されており、機械検証に適している |
| **レガシーからの決別** | POSIX・Win32・ELFの前提なし。互換性は将来のオプトインレイヤーであり、制約ではない |
| **デフォルトの観測可能性** | すべてのケイパビリティ付与・モデル呼び出し・メモリ移動・ポリシーオーバーライドは監査可能なイベントである |
| **異種リソースオーケストレーション** | CPU・GPU・NPU・VRAM・DRAM・ストレージIOPS・ネットワークキューが第一級のスケジューリング対象資源 |
| **人間と機械の調和** | OSはAIエージェントと人間ユーザーの両方を、共有されたインテント中心のインタフェースモデルで支援する |
| **AI主導開発** | LLMが設計・実装の中心を担い、人間はアーキテクチャレビュー・安全境界・不可逆な意思決定の最終権限者となる |

## 高レベルアーキテクチャ

Reikan は、小さな検証済みマイクロカーネルコアと高性能システムサービスレイヤーの二層構造で構成されます：

```
┌─────────────────────────────────────────────────────┐
│        人間向けUI / エージェントシェル / 開発環境       │
├─────────────────────────────────────────────────────┤
│  モデルランタイム │ オブジェクトストア │ NWメッシュ │ …  │
│                システムサービスレイヤー                 │
├─────────────────────────────────────────────────────┤
│  ケイパビリティ管理 │ スケジューラ │ VMマネージャ │ IPC │
│             検証済みマイクロカーネルコア               │
├─────────────────────────────────────────────────────┤
│   CPU  │  GPU  │  NPU/TPU  │  DMA  │  ストレージ    │
│              ハードウェアリソースファブリック             │
└─────────────────────────────────────────────────────┘
```

主要な抽象は **エージェント・ケイパビリティ・コンテキスト・メモリオブジェクト・実行グラフ・モデルリソース** であり、プロセス・ファイル・スレッドではありません。
